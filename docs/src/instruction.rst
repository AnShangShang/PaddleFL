Quick Start Instructions
========================

Install PaddleFL
-----------
To install Paddle Federated Learning, we need the following packages.


.. code-block:: sh

    paddlepaddle >= 1.6 (Faster performance on 1.6)
    networkx
    cython

We can simply install PaddleFL by pip.

.. code-block:: sh

    pip install paddle_fl
